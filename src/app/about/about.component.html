About this application


Like I have said in footer.. Game Score is an interactive learning project created by me (Otto) using the game codes that I have found in Internet like a base (I mean, I have modificate these codes and added other things).

Until this moment.. I have used these technologies in this project:

- Angular (Phaser 3 and Canvas);

- NodeJs (Express, JWT and Mongoose);

- SpringBoot (JWT and others Spring frameworks like Spring Jpa, Spring Security, Spring Webflux and etc...);

- React for Admin ui.. not started yet;

- Mongo Db;

- Heroku.
